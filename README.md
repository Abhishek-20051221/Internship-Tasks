Responsive Company Landing Page

Description:

This project is a simple and responsive landing page built using HTML5 and CSS3.
It includes a header with navigation, a hero section with a call-to-action button, and a footer with social media links.

The layout is built using Flexbox and is fully responsive using CSS media queries.

Technologies Used:

1.HTML5

2.CSS3

3.Flexbox

4.Media Queries
5.VS Code
6.Live Server

Features:

1.Responsive design (mobile, tablet, desktop)
2.Clean and minimal UI
3.Header with navigation links
4.Hero section with CTA button
5.Footer with social media links
6.Consistent color theme
7.Hover effects for interactive elements

Responsiveness:

1.Navigation adjusts on smaller screens
2.Layout stacks properly on mobile devices
3.Tested using browser resize and developer tools

How to Run:

1.Download or clone the project
2.Open the folder in VS Code
3.Right-click index.html
4.Select Open with Live Server
5.Resize the browser to test responsiveness

Objective
The objective of this project is to demonstrate understanding of:

1.Semantic HTML structure
2.CSS layout using Flexbox
3.Responsive design principles
4.Clean UI styling
